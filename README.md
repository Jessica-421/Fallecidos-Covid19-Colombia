# Fallecidos-Covid19-Colombia
Análisis de fallecimientos por COVID-19 en Colombia, con limpieza de datos, visualizaciones y gráficos interactivos usando Python.

## Contenido del proyecto
- Limpieza y normalización de datos
- Verificación de datos duplicados
- Eliminación de columnas con muchos valores nulos
- Análisis por departamento y sexo
- Gráficos interactivos de tendencias temporales con Plotly

## Cómo usar
1. Abrir el notebook `Análisis_fallecidos_Covid19_Colombia` en Google Colab, Jupyter o Visual Studio
2. Ejecutar las celdas para reproducir los análisis y gráficos
